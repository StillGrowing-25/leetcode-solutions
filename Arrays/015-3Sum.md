# 3Sum

Problem Link: https://leetcode.com/problems/3sum/

# Intuition

A brute-force approach would check every possible triplet using three nested loops, resulting in **O(n³)** time complexity. Since we only need unique triplets whose sum is zero, we can optimize the search by first sorting the array and then using the **Two Pointers** technique.

Sorting allows us to efficiently adjust the sum by moving pointers and also makes it easy to skip duplicate elements, ensuring that only unique triplets are added to the answer.

# Approach

1. Sort the given array in non-decreasing order.
2. Iterate through the array, fixing one element `nums[i]` at a time.
3. Skip duplicate values of `nums[i]` to avoid repeated triplets.
4. Initialize two pointers:
   - `j = i + 1` (left pointer)
   - `k = n - 1` (right pointer)
5. While `j < k`:
   - Compute the sum `nums[i] + nums[j] + nums[k]`.
   - If the sum is `0`:
     - Store the triplet.
     - Move both pointers inward.
     - Skip duplicate values of `nums[j]`.
   - If the sum is greater than `0`, decrement `k` to reduce the sum.
   - Otherwise, increment `j` to increase the sum.
6. Continue until all possible unique triplets are found.

# Complexity

- **Time complexity:** **O(n²)**
  - Sorting takes **O(n log n)**.
  - The outer loop runs **n** times, and the two pointers together traverse the array only once for each iteration.
  - Overall complexity: **O(n²)**.

- **Space complexity:** **O(1)** (excluding the output array)
  - Only a few extra variables are used.
  - The returned answer is not counted in auxiliary space.

# Code

```cpp
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        sort(nums.begin(), nums.end());

        vector<vector<int>> ans;
        int n = nums.size();

        for (int i = 0; i < n; i++) {
            if (i > 0 && nums[i] == nums[i - 1])
                continue;

            int j = i + 1;
            int k = n - 1;

            while (j < k) {
                int sum = nums[i] + nums[j] + nums[k];

                if (sum == 0) {
                    ans.push_back({nums[i], nums[j], nums[k]});
                    j++;
                    k--;

                    while (j < k && nums[j] == nums[j - 1])
                        j++;
                }
                else if (sum > 0) {
                    k--;
                }
                else {
                    j++;
                }
            }
        }

        return ans;
    }
};
